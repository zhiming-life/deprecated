# 示例对话

终端 demo 的流程示意（解读内容随你接的模型而变，这里只示意**形状**）。

```
你 › 看看我的运势
师傅 › 好啊，先报生辰——年月日和大概时辰，公历农历都行。
你 › 1990年6月15号下午两点，公历
师傅 ›
你外柔内刚，待人温和……
日主辛金生于午月，火旺当令……
事业宜靠专业立身（把握高）；婚姻大方向平顺，时点再看（把握中）……
你 › q
　后会有期。
```

发生了什么：

1. supervisor 缺生辰 → 追问，不惊动宗师。
2. 收齐五项 → 路由八字（齐没齐由编排层按字段判定，不靠模型自报）。
3. core.paipan() 把生辰排成四柱（Python 算死）。
4. bazi_master 在盘面上分段解读，每条标把握程度。

全部串在 `zhiming_life/conductor.py` 一个文件里。RAG 默认返回空，联想段的引证取决于模型自身知识；接上语料后才可追溯。
